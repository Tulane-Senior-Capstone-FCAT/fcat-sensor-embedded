Memory Layout:

Total: 512 bytes

The first 8 bytes:
2 integers
1: Measurement count
2: Unknown Currently

Measurement             (22 bytes):
Moisture: 2 bytes       (unsigned int)
Temperature: 8 bytes    (double)
Humidity: 8 bytes       (double)
Light: 2 bytes          (unsigned int)
Battery: 2 bytes        (unsigned int)
